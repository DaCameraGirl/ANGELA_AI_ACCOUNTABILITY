## Contribution Standard

Useful contributions to this repository would include:

- tighter chronology
- clearer source classification
- technical clarification without overclaim
- accessibility improvements
- better distinctions between artifact, interpretation, and overclaim

Contributions that weaken the repository are the opposite:

- flattening Angela's authorship
- overstating what the evidence proves
- erasing uncertainty where uncertainty is real
- treating source artifacts as if they were already final conclusions
