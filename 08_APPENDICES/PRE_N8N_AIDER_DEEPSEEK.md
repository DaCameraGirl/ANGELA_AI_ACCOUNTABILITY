# Pre-N8n: Aider, DeepSeek, and the June Setup Layer

**Purpose:** preserve the mailbox evidence that predates the stronger workflow narrative  
**Why it matters:** the repo frames `n8n` as the main origin platform, but the mailbox shows earlier `aider` and `DeepSeek` activity in June 2025

This appendix is about sequence, not theory.

## Best current date anchors

The mailbox currently supports this order:

1. **June 18, 2025:** `aider` command using a `DeepSeek` model
2. **June 20, 2025:** self-email about getting **Aider Chat** running on Windows
3. **June 25, 2025:** self-mail tied to **DeepSeek**
4. **June 27 to June 30, 2025:** clear **n8n** account and workspace activity
5. **July 2025 onward:** stronger workflow, emergence, and escalation framing

That means the early sequence is not:

- `n8n first, everything else later`

It is closer to:

- `aider + DeepSeek first`
- `n8n later in June`
- `larger workflow story after that`

So `n8n` is definitely part of the June layer too, just not the earliest visible part of it.
